The Eisenhower Matrix, also known as the Urgent-Important Matrix, is a time management tool designed to help prioritize tasks based on their urgency and importance. This method was inspired by Dwight D. Eisenhower, who famously said:

"What is important is seldom urgent, and what is urgent is seldom important."

How It Works

The matrix divides tasks into four quadrants:

1. Urgent and Important (Do First)

  • Critical tasks requiring immediate attention.
  • Examples: Deadlines, emergencies, pressing issues.
  • Action: Complete these tasks as soon as possible.

2. Not Urgent but Important (Plan)

  • Essential for long-term success, but not immediate.
  • Examples: Strategic planning, skill-building, exercise.
  • Action: Schedule these tasks to ensure they're done.

3. Urgent but Not Important (Delegate)

  • Demand immediate attention but don't contribute significantly to goals.
  • Examples: Interruptions, minor requests, certain meetings.
  • Action: Delegate these tasks to someone else if possible.

4. Not Urgent and Not Important (Eliminate)

  • Distractions that don't add value.
  • Examples: Excessive social media, trivial activities.
  • Action: Minimize or eliminate these tasks.

Visual Representation

The matrix is structured as a 2x2 grid:

Urgent Not Urgent
Important Do First (Quadrant 1) Plan (Quadrant 2)
Not Important Delegate (Quadrant 3) Eliminate (Quadrant 4)

Benefits

  • Helps prioritize tasks effectively.
  • Reduces stress by focusing on what truly matters.
  • Encourages long-term planning.
  • Eliminates unnecessary tasks.

Tips for Effective Use

  • Regularly review tasks to ensure correct categorization.
  • Focus on Quadrant 2 tasks to prevent crises.
  • Delegate (Quadrant 3) and say no to (Quadrant 4) tasks.

Hot Keys

  • Mac: Press Ctrl + Alt + T to show the app
  • Windows: Press Command + Option + T to show the app

Features

  • Add Task: Click "Add Task" on the main page. Enter the task name, description (optionally using markdown), select urgent/important if applicable, set a deadline, and click "Add Task" to save. Click "Back" to return to the main page.
  • Edit Notes: Click "View Task" then "Edit" to write notes in markdown. The preview updates automatically. Click the provided Markdown guide link for help. After editing, click "Save Notes" to save and then "Back" to return.
  • Delete Task: Check the box next to any task(s) you wish to remove, then click "Delete Selected". Once deleted, these tasks cannot be recovered.
